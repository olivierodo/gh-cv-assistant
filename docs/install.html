Thank you for installing the gh-cv-assistant

Since the Github App is installed on your repository let's ensure that you have everything setup to empower your resume



1. The github app can't be install on an organisation

Be ensure that you install the Github app on one repository at the time, it can't handle installation on organisation

2. Ensure that you repository is public

The Github app is using the API from latexonline.cc. To let the website latexonline access to your repository, it needs to be public

3. Create the config file .github/cv-assistant.yml

To know which document compile, the Github App is reading the file .github/cv-assistant.yml that is on your repository. An example here

4. Master branch is the source of truth

The Github app will generate the documents only when some changes get triggered but the master branch.

5. Release creation

The Github app will create 2 releases : latest and vYYMMDD.HH.mm. The generated document will be upload in the releases. The latests release documents will be updated automatically at each new update.

6. Communication

You might face some trouble during the installation or during the generation of the pdf.
However the Github App will open some issue on your repository to ensure you are aware of any problem
